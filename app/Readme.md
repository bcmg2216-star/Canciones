# Aplicación de canciones

## Descripción
Esta aplicación muestra un listado de canciones en tarjetas con su imagen, título, artista, álbum y duración.
Permite:
- Visualizar las canciones, borrar canciones y tienes un botón de añadir pero todavía no lo hemos implementando.

## Tipo de aplicación y planes futuros
La aplicación será un gestor de aplicaciones (tipo MP3 o Spotify) donde los usuarios podrán ver un listado de aplicaciones con su información. 
En el futuro implementaré opciones para añadir y actualizar canciones.
